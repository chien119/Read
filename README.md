# 功能說明:
  音樂遊戲，當音符落到白色判定線時按下對應的按鍵來消除音符，若超過判定線則喪失一條生命，判定線往上兩格為得分範圍，在音樂結束前還有剩餘生命為過關，反之則失敗，結束後按任意鍵可重新開始。
